# CaloriesPred | AI-Powered Calories Estimation
![Screenshot](/calorie-prediction.png)

## Project Manifesto

### Introduction
**CaloriesPred** adalah aplikasi web berbasis **AI** yang dirancang untuk memperkirakan jumlah kalori yang terbakar selama aktivitas fisik. Perhitungan dilakukan menggunakan **algoritma Random Forest** berdasarkan berbagai parameter input seperti durasi, usia, berat badan, tinggi badan, jenis kelamin, suhu, dan detak jantung (**BPM**).

Proyek ini dikembangkan sebagai **tugas akhir untuk mata kuliah Machine Learning**, dengan tujuan memadukan pembelajaran algoritma prediksi dan pengembangan aplikasi web yang responsif.
Aplikasi ini dapat digunakan untuk kebutuhan kebugaran pribadi, pelatihan atlet, maupun pengelolaan kesehatan secara umum.
