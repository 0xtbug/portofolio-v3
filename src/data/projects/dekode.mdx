# DeKode | Lambda Encoded String Decoder
![Screenshot](/dekode.gif)

## ğŸ“Œ Introduction
**DeKode** adalah tools yang dirancang untuk mendekode string yang telah diencode menggunakan metode lambda, seperti yang dihasilkan oleh [PyObfuscate](https://github.com/htr-tech/PyObfuscate).
Tools ini memudahkan proses decoding dengan dukungan berbagai metode umum berbasis **Base64**, **Base16 (B16)**, **Base32 (B32)**, dan **Zlib**.

---

## ğŸŒŸ Key Features
- Mendekode string dengan metode:
  - `Lambda base64.b16decode`
  - `Lambda base64.b32decode`
  - `Lambda base64.b64decode`
  - `Lambda zlib.decompress(base64.b16decode)`
  - `Lambda zlib.decompress(base64.b32decode)`
  - `Lambda zlib.decompress(base64.b64decode)`
