# Kamus Indonesia-Sunda
![Preview](/kamus-indo-sunda.png)

## Project Manifesto
### Introduction
Proyek ini merupakan hasil dari **tugas akhir mata kuliah Analisa Desain Algoritma**, yaitu sebuah aplikasi kamus dua bahasa (Indonesia-Sunda) yang dibangun menggunakan **Python** dan **Flask**, serta memanfaatkan **algoritma Binary Search Tree (BST)** untuk menyimpan, mencari, dan mengelola kata dalam kamus.

---

## 🌟 Feature
- Tambah Kata
- Hapus Kata
- Cari Terjemahan
- Lihat Semua Kata
