## Pendahuluan
**Bookmark Manager** adalah client web modern untuk [Linkding](https://github.com/sissbruecker/linkding).
Dengan aplikasi ini, kamu bisa **menjelajahi, mencari, memfilter, dan mengelola semua bookmark yang tersimpan** melalui antarmuka yang bersih dan responsif.
Dibangun dengan **Next.js 15, React 19, dan Tailwind CSS**, aplikasi ini menghadirkan pengalaman cepat, mobile-friendly, dan terintegrasi mulus dengan API Linkding.

### Prasyarat
- Node.js 18+
- pnpm (disarankan) atau npm
- Instance Linkding yang sudah berjalan

### Langkah
```bash
# Clone repository
git clone https://github.com/0xtbug/bookmark-manager.git
cd bookmark-manager

# Install dependencies
pnpm install
# atau
npm install

# Konfigurasi environment
cp .env.example .env.local
```

Edit file `.env.local`:

```env
LINKDING_API_URL=http://alamat-linkding-anda.com
LINKDING_API_TOKEN=token-api-anda
```

Jalankan server development:

```bash
pnpm dev
# atau
npm run dev
```
